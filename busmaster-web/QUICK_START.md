# 🚀 Quick Start - BUSMASTER Web

**Get up and running in 5 minutes!**

---

## Prerequisites

- **Node.js 20+** (https://nodejs.org/)
- **Git** (https://git-scm.com/)

---

## 1. Install Dependencies

```bash
# Backend
cd server
npm install

# Frontend (new terminal)
cd web-ui
npm install
```

---

## 2. Start Servers

**Terminal 1 - Backend:**
```bash
cd server
npm run dev
```
✅ API running at: http://localhost:8080

**Terminal 2 - Frontend:**
```bash
cd web-ui
npm run dev
```
✅ UI opens at: http://localhost:3000

---

## 3. Try It Out!

1. **Dashboard** - See overview
2. **Configuration** - Select "Virtual CAN" and click "Connect"
3. **Message Window** - Watch simulated CAN messages flowing!
4. **Transmit** - Enter message ID and data to send
